# Why?

This sample is to remember how to do implicit type casting.

# What?

## Build

```
gradle clean build
```

_Note: Text_

## Install

```
gradle install
```

_Note: Text_

## Run

```
gradle run -Dmyvariable=myvalue -Dexec.args="myarg1 myarg2"
```
